Product Requirements Document (PRD)
Project Title

Fleet Tracker Lite – Vehicle Tracking & Trip Management System

Version: 1.0

Project Type: Full Stack Web Application

Backend: Rust (Axum)

Frontend: Next.js + Tailwind CSS

Database: PostgreSQL

Maps: Leaflet + OpenStreetMap

Authentication: JWT

1. Executive Summary

Fleet Tracker Lite is a web-based fleet management application that enables transport companies to manage trucks, drivers, and trips while simulating real-time vehicle tracking on a map.

The application is designed for educational purposes and portfolio demonstrations. It simulates GPS data without requiring physical GPS hardware, allowing the entire system to run on a single laptop.

2. Problem Statement

Small and medium transport companies often struggle with:

Tracking active trips
Managing trucks and drivers
Monitoring trip progress
Viewing vehicle locations
Generating operational reports

Fleet Tracker Lite provides a centralized platform to simplify these operations.
